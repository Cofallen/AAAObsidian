> [!abstract] 定义与本源
> **解析定义**
> 双曲正弦与双曲余弦由指数函数唯一定义：
> $$\sinh x = \frac{e^x - e^{-x}}{2}, \qquad \cosh x = \frac{e^x + e^{-x}}{2}.$$
> 这一对称构造，使它们天然继承了指数函数的强大性质。
>
> **几何定义：为什么叫“双曲”？**
> 类比三角函数，即“单位圆函数”：点 $(\cos t, \sin t)$ 落在单位圆 $x^2 + y^2 = 1$ 上。
> 双曲函数的点 $(\cosh t, \sinh t)$ 则落在**等轴双曲线** $x^2 - y^2 = 1$ 的右支上：
> $$\cosh^2 t - \sinh^2 t = 1.$$
> 其中，变量 $t$ 表示由双曲线、$x$ 轴及原点到该点的连线所围成区域面积的**两倍**。这与圆函数中 $t$ 表示弧长或扇形面积的**两倍**，构成了绝妙对偶。

> [!info] 名字的故事与历史
> **“双曲”(Hyperbolic)** 一词，正是源于它们用来参数化双曲线 (Hyperbola)。而 **“正弦”(Sine)** 与 **“余弦”(Cosine)** 的命名，则直接借用了三角函数，因其满足极其相似的恒等式体系，仅有些许符号差异。
>
> **历史脉络**
> - **悬链线的解答 (1690年)**：雅各布·伯努利向欧洲数学界发起挑战，求解一根柔软且密度均匀的绳子自然下垂时的形状。伽利略曾误认为它是抛物线。最终，约翰·伯努利、莱布尼茨和惠更斯几乎同时证明，这条曲线正是双曲余弦的图象 $y = a\cosh(\frac{x}{a})$。
> - **18世纪，文森佐·里卡蒂与兰伯特**：意大利数学家文森佐·里卡蒂首次系统地研究了这类函数，并使用了 $\sinh$ 和 $\cosh$ 的记号。随后，约翰·海因里希·兰伯特为其奠基，证明了反双曲函数可由自然对数表达，并广泛推广了这些函数，使其成为分析学中不可或缺的工具。

> [!important] 汇总：核心性质与恒等式
> **奇偶性**
> $$\sinh(-x) = -\sinh x \quad (\text{奇函数}), \qquad \cosh(-x) = \cosh x \quad (\text{偶函数}).$$
> **值域与单调性**
> $$\sinh x: \mathbb{R}\to\mathbb{R}, \text{ 单调递增.}$$
> $$\cosh x: \mathbb{R}\to[1,+\infty), \text{ 在 }(-\infty,0] \text{ 递减，在 }[0,+\infty) \text{ 递增，最低点 } (0,1).$$
> **渐近行为**
> $$x\to +\infty: \ \sinh x \sim \cosh x \sim \frac12 e^x; \qquad x\to -\infty: \ \sinh x \sim -\frac12 e^{-x},\ \cosh x \sim \frac12 e^{-x}.$$
> **平方恒等式 (最重要)**
> $$\cosh^2 x - \sinh^2 x = 1.$$
> **和角与倍角**
> $$\sinh(x\pm y) = \sinh x \cosh y \pm \cosh x \sinh y,$$
> $$\cosh(x\pm y) = \cosh x \cosh y \pm \sinh x \sinh y.$$
> $$\sinh 2x = 2\sinh x \cosh x, \quad \cosh 2x = \cosh^2 x + \sinh^2 x.$$

> [!tip] 导数与泰勒展开
> **导数 (完美循环)**
> $$(\sinh x)' = \cosh x, \qquad (\cosh x)' = \sinh x.$$
> **泰勒展开 (在 $x=0$ 处，所有项均为正)**
> $$\sinh x = x + \frac{x^3}{3!} + \frac{x^5}{5!} + \cdots = \sum_{n=0}^\infty \frac{x^{2n+1}}{(2n+1)!},$$
> $$\cosh x = 1 + \frac{x^2}{2!} + \frac{x^4}{4!} + \cdots = \sum_{n=0}^\infty \frac{x^{2n}}{(2n)!}.$$

> [!info] 反双曲函数
> 由定义可解出反函数，在积分中极为常用。
> $$\operatorname{arcsinh} x = \ln\left(x + \sqrt{x^2+1}\right), \quad x\in\mathbb{R}.$$
> $$\operatorname{arccosh} x = \ln\left(x + \sqrt{x^2-1}\right), \quad x\ge 1.$$
> 导数简洁优美，对应两个核心积分公式：
> $$(\operatorname{arcsinh} x)' = \frac{1}{\sqrt{x^2+1}}, \qquad (\operatorname{arccosh} x)' = \frac{1}{\sqrt{x^2-1}}.$$

> [!question] 考研典型例题
> **例** 求极限 $\displaystyle \lim_{x\to 0}\frac{\cosh x - 1}{\sinh x - x}$.
> 
> **解** 利用泰勒展开。当 $x\to0$ 时，
> $$\cosh x = 1 + \frac{x^2}{2} + \frac{x^4}{24} + O(x^6), \quad \sinh x = x + \frac{x^3}{6} + O(x^5).$$
> 代入极限式：
> $$\frac{(1+x^2/2+\cdots)-1}{(x+x^3/6+\cdots)-x} = \frac{x^2/2 + o(x^2)}{x^3/6 + o(x^3)} = \frac{3}{x} \to \infty.$$
> 故极限不存在（趋于无穷大）。这体现了虽然 $x\to0$ 时 $\sinh x \sim x$，但 $\sinh x - x \sim \frac{x^3}{6}$，不能直接等价于 $0$。

> [!warning] 常见错误
> 1. **导数符号**：$(\cosh x)' = \sinh x$，没有负号。
> 2. **平方恒等式**：是 $\cosh^2 x - \sinh^2 x = 1$，是减号。
> 3. **反函数定义域**：$\operatorname{arccosh} x$ 要求 $x\ge 1$，且其值域为 $[0,+\infty)$。
> 4. **极限误判**：$\sinh x - x$ 在 $x\to0$ 时的等价无穷小是 $\frac{x^3}{6}$。
