> [!abstract] 欧拉常数的定义
> 欧拉常数 $\gamma$ 定义为调和级数与自然对数之差的极限：
> $$\gamma = \lim_{n\to\infty} \left( 1+\frac12+\frac13+\cdots+\frac1n - \ln n \right).$$
> 其近似值约为 $0.5772156649\dots$，它是一个数学常数，但至今尚不清楚它是否为无理数。

> [!important] 存在性证明 (考研重点)
> 证明数列 $x_n = 1+\frac12+\cdots+\frac1n - \ln n$ 收敛，核心是**单调有界准则**。
> 
> **有界性**：利用不等式 $\frac{1}{n+1} < \ln\left(1+\frac1n\right) < \frac1n$，可得
> $$x_n > \sum_{k=1}^n \left[\ln\left(1+\frac1k\right)\right] - \ln n = \ln(n+1) - \ln n > 0.$$
> 另一方面，
> $$x_n < 1 + \sum_{k=2}^n \left[\ln\left(1+\frac{1}{k-1}\right)\right] - \ln n = 1.$$
> 故 $0 < x_n < 1$，数列有界。
> 
> **单调性**：计算
> $$x_{n+1} - x_n = \frac{1}{n+1} - \ln\left(1+\frac1n\right) < 0.$$
> 故数列严格单调递减。由单调有界准则，极限存在，此极限即为 $\gamma$。

> [!tip] 考研常用展开式
> 调和级数的渐近公式：
> $$H_n = 1+\frac12+\cdots+\frac1n = \ln n + \gamma + \frac{1}{2n} - \frac{1}{12n^2} + O\left(\frac{1}{n^4}\right).$$
> 这常用于求某些含调和数极限的精确值，如：
> $$\lim_{n\to\infty} n\left(H_n - \ln n - \gamma\right) = \frac12.$$

> [!info] 与 Gamma 函数的关系
> 欧拉常数是 Gamma 函数在 $1$ 点处对数导数的负值：
> $$\gamma = -\Gamma'(1) = -\int_0^\infty e^{-x}\ln x\,dx.$$
> 这也给出了它的积分表达式：
> $$\gamma = -\int_0^\infty e^{-x}\ln x\,dx.$$

> [!warning] 易错提醒
> 1. 欧拉常数 $\gamma$ 与自然常数 $e$ 是完全不同的两个数，切勿混淆。
> 2. 判断级数 $\sum_{n=1}^\infty \left( \frac1n - \ln\left(1+\frac1n\right) \right)$ 的敛散性时，正好用比较判别法的极限形式，其和为 $\gamma$。
